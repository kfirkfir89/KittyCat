KITTYCAT: A Coding Project

When it counts you KITTY CAT. A javascript coding exercise using loops. Write a program that prints the numbers from 1 to 100 and multiples of '3' display "KITTY" instead of number and for the multiples of '5' display "CAT" multiple of both display KITTYCAT.

Instructions

Enter a kitty and cat value. Press the button and the values will appear.
You can also put in values from 1 to 9 to see your KITTYCAT matchs.
